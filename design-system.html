Glassmorphism

Advanced frosted glass UI components with Display-P3 color space, layered gradients, and backdrop blur effects.

Glass Card

Light card

Uses the .card-glass class with @include glassmorphism-light mixin for subtle frosted glass effect with light backgrounds.

Dark card

Uses the .card-glass--dark class with @include glassmorphism-dark mixin for darker frosted glass effect with mix-blend-mode: plus-lighter.

Technical implementation details

The glassmorphism effect is defined in _sass/abstracts/_mixins.scss as a reusable mixin: 

@mixin glassmorphism-light($blur: 5px) {
  // 5 layered gradients with Display-P3 color space
  background: linear-gradient(...);
  background-blend-mode: plus-lighter, plus-lighter,
                         normal, color-burn, overlay;

  // Inset shadows + drop shadow
  box-shadow: 16px 16px 9px -18px color(display-p3 1 1 1) inset,
              -12px -12px 6px -14px color(display-p3 0.88 0.88 0.88) inset,
              ...;

  // Backdrop blur
  backdrop-filter: blur($blur);
  -webkit-backdrop-filter: blur($blur);

  transition: all var(--transition-base);
}

@mixin glassmorphism-dark($blur: 5px) {
  // Same properties as glassmorphism-light
  // Used for dark variant glass cards
}

Usage in SCSS 

.my-glass-element {
  @include glassmorphism-light;      // Default 5px blur
  @include glassmorphism-light(10px); // Custom blur amount
}

.my-dark-glass-element {
  @include glassmorphism-dark;
}
