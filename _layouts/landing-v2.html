How we bring compute into the open

Caution combines three core technologies so you can verify software from the toolchain that builds it to the hardware that runs it.

Integrity

Reproducible builds

Builds that are bit for bit identical from the same source. They ensure software integrity, catch supply chain tampering, and make opaque systems verifiable.

Isolation

Secure enclaves

Hardware isolated execution that keeps external processes from touching sensitive workloads. Enclaves provide strong isolation, but isolation alone cannot prove what actually ran.

Proof

Remote attestation

Hardware attestation reports the measured state of a machine and gives cryptographic proof of what is running. It anchors trust in hardware, but does not verify how the software was built.
